# srss4docker
Try to run plain old JavaScript at NodeJS inside a Docker container.

# On CLI:

## Build Docker Image from Dockerfile

    docker build -t nana  .

## Run the image in a container instance

    docker run nana

## WSL - Allow installation of Windows-Subsystem für Linux


    Windows-Subsystem für Linux 2.7.3


    https://aka.ms/wslinstall

    wsl.exe --update
